# Matrix Maze

A cheerful, dependency-free maze game for children. Guide the explorer from **IN** to **OUT**, celebrate together, and continue into a new random matrix.

## Run it

Open `index.html` in a browser, or serve the folder with any simple static server.

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- A new random maze after every completion
- Mouse-hover movement without clicking and finger-drag controls
- A colored route that shows and remembers the current path
- Friendly movement notes and a completion melody using the Web Audio API
- Confetti, encouragement, and team-focused celebration messages
- Progressively larger matrices
- Responsive layout for desktop, tablet, and mobile
